<title>Definicion de Arreglos</title>

Notion: https://www.notion.so/DS-Array-4863672efa754f4c9f84802a8ee4d47d?pvs=4

Un arreglo en Java es una estructura de datos que permite almacenar varios valores o elementos de un mismo tipo en una única variable.

<ul>
    <li>Almacenamiento eficiente: Los arreglos permiten almacenar datos de manera contigua en la memoria, lo que facilita el acceso rápido a los elementos.</li>
    <li>Acceso indexado: Cada elemento en un arreglo puede ser accedido directamente mediante su índice, lo que hace muy eficiente la lectura y la modificación de elementos.</li>
    <li>Facilidad de iteración: Los arreglos pueden ser fácilmente recorridos utilizando bucles, permitiendo realizar operaciones sobre cada uno de sus elementos.</li>
</ul>
