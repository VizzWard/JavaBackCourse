<title>Tipos de datos en Arreglos</title>

Los arreglos pueden contener cualquier tipo de dato, incluyendo números enteros, números de punto flotante, caracteres y objetos.
El tipo de dato del arreglo define el tipo de elementos que el arreglo puede almacenar.
Por ejemplo, un arreglo de tipo int sólo puede almacenar números enteros, mientras que un arreglo de tipo String puede almacenar cadenas de texto.

<ul>
    <li>Arreglos de tipos primitivos: Son arreglos que almacenan tipos de datos primitivos como int, double, char, etc. Son muy eficientes en términos de memoria y rendimiento.</li>
    <li>Arreglos de objetos: Pueden almacenar objetos de cualquier clase, incluyendo las clases envolventes de tipos primitivos como Integer, Double, Character, etc., y también arreglos que contienen instancias de clases definidas por el usuario.</li>
</ul>

La elección del tipo de dato para un arreglo depende del tipo de datos que necesitas almacenar y cómo planeas usar esos datos en tu programa.
